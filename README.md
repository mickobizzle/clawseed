# ClawSeed

A public starter kit for building your own "claw": an agentic wrapper around LLMs with local automation and safety-first workflows.

## Includes
- Onboarding questionnaire and defaults
- Agent policy templates
- Weekly review checklist (wrapper-agnostic)
- Safety/redaction guidance

## Does NOT include
- Personal knowledge bases
- Topic hubs
- Zettelkasten corpus
- Private logs, inboxes, or identifiers
